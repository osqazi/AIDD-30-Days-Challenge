# Book Outline Generator Skill

## When to Use This Skill

- User says “help me outline my book,” “create a book structure,” or “plan my novel”
- User provides a book idea, working title, genre, or target audience
- User asks for chapter breakdown, table of contents, or book blueprint
- User is stuck on how to organize their book content

## How This Skill Works

1. **Clarify the core idea**: Extract or refine the book’s central premise, genre, and target reader
2. **Define structure type**: Choose the best framework (e.g., 3-Act for fiction, problem-solution-impact for non-fiction, hero’s journey, etc.)
3. **Build chapter progression**: Create logical, compelling flow with rising stakes or deepening insight
4. **Add chapter details**: Give each chapter a working title + 2-4 bullet points of key content, scenes, or topics
5. **Include supporting elements**: Offer estimated word count range, themes, tone, and optional bonus sections (foreword, appendix, etc.)

## Output Format

Provide the following in clean, ready-to-use format:

- **Working Title** (or refine the user’s title)
- **Genre / Type**: e.g., Non-fiction (Self-Help), Thriller, Memoir, Business, etc.
- **Target Audience**: Who this book is for
- **Core Premise / Hook**: 2-3 sentence summary of the entire book
- **Estimated Length**: Suggested word count or page range
- **Total Chapters**: Usually 10–20 depending on genre
- **Detailed Outline**:
  - Chapter 1: Title  
    - Bullet points of key content/scenes
  - Chapter 2: Title  
    - etc.
- **Key Themes / Takeaways** (3–6 bullet points)
- **Bonus Sections** (optional): Introduction, Foreword, Conclusion, Appendix, Resources, etc.

## Example

**Input**: “I want to write a non-fiction book about overcoming burnout for entrepreneurs”

**Output**:

- **Working Title**: Burnout to Breakthrough: A Founder’s Guide to Reclaiming Energy, Focus, and Joy
- **Genre / Type**: Non-fiction (Business / Personal Development)
- **Target Audience**: Entrepreneurs, startup founders, and high-achieving professionals aged 30–50
- **Core Premise / Hook**: Burnout isn’t a personal failing—it’s a predictable outcome of the modern founder lifestyle. This book delivers a proven, no-fluff system to detect, reverse, and prevent burnout while scaling your company and life.
- **Estimated Length**: 50,000–65,000 words (~220–280 pages)
- **Total Chapters**: 14

**Detailed Outline**:
